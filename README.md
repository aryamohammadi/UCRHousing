# DormDuos

A housing platform for UCR students to find and post off-campus listings.

## Tech Stack
React, Node.js, Express, MongoDB, JWT, Jest, Vercel, Railway

## Features
- Search and filter 10k+ listings by price, bedrooms, amenities
- JWT authentication with bcrypt password hashing
- Input sanitization to prevent NoSQL injection
- 100% test coverage with Jest and Supertest

## Live Demo
[dormduos.com](https://dormduos.com)

## Run Locally
```bash
git clone https://github.com/aryamohammadi/Dormduos.git
cd Dormduos/backend && npm install && npm run dev
cd ../frontend && npm install && npm run dev
```
